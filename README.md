## Running
1. Install the requirements.txt with pip (base command: `pip install -U -r requirements.txt`
2. Configurate the variables in main.py (TOKEN, GOOGLE_API_KEY, and LOG_CHANNEL_ID)
3. Run main.py `python3 main.py`
